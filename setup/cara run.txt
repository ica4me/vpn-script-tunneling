Persyaratan VPS (Hardware)
Agar proses instalasi lancar (terutama saat proses compile Haproxy di 6.sh dan Python di 1.sh):

RAM: Minimal 1 GB (Sangat disarankan). Jika RAM hanya 512 MB, proses kompilasi mungkin akan gagal karena kehabisan memori.

Disk Space: Minimal sisa 5 GB.

Kondisi OS: Harus Fresh Install (OS yang baru di-reinstall). Jangan menjalankan script ini di OS yang sudah terpasang web server atau VPN lain karena akan terjadi konflik port.
